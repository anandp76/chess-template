    Event Description

    Hide Online Username:

    If checked, then the online user name is not shown on the entry list.

    Team Required:

    If checked, then the school/team name field must be filled in and cannot be left blank. This only applies to Individual events with teams.

    FIDE Rated:

    Check this box if one or more sections are FIDE rated.

    Request Grade:

    Prompt for school grade if one or more sections are restricted by scholastic grade level.

    Request Referral:

    Prompt for a referred by name and/or email address.

    Request Address:

    Prompt for the mailing address.

    Allow Byes:

    If unchecked, the bye selection is removed from the registration page.

    Prompt For Donation:

    If checked, the user is given an option to donate the specified amount to cover transaction costs. $2 or $3 is suggested for most events.
